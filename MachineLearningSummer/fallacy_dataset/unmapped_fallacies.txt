Define <I> to be a defective argument type declaring a rare occurence as impossible (e.g., "No one can fall in love at the first sight.", "No one ever wins the lottery; it's rigged.", "It's impossible for anyone to run a mile in under four minutes.")
Define <IE> to be a defective argument type failing to list all possibilities in a proposition (e.g., "The victim was either attacked by robbers, enemies, or you," without considering other potential suspects.", "Trees are either oaks or pines," ignoring the multitude of other tree species, "Energy sources are just coal or solar," ignoring wind, nuclear, hydro, etc.)
Define <RT> to be a defective argument type tracing the origin of an argument too far back unnecessarily (e.g., unnecessary elaboration on a well-understood cause, "The pollution problem can be traced back entirely to the Industrial Revolution.", "Our employee turnover goes back to a decision made ten years ago.")
Define <GR> to be a defective argument type providing a false reason that doesn’t uphold scrutiny (e.g., "Philosophy should be avoided because it results in sloth," a contested assertion, "Studying history is pointless because it fills your head with useless dates.", "Investing in stocks is useless because the market always crashes.")
Define <MR> to be a defective argument type using the same reason for different propositions without proper justification (e.g., Pacuvius using the same reason for different qualities, using the same argument, "because it leads to happiness," to justify both drinking coffee and exercising, "Because it saves money," used to support both cooking at home and using public transit)
Define <DM> to be a defective argument type providing dilemmas that can be easily reversed or rebutted (e.g., "If you think X is evil, why did you...?", "If flying is safe, why do people fear flying?", "If you believe in science, why not accept all scientific claims without question?")
Define <MOS> to be a defective argument type misusing signs that represent a variety of things to indicate one specific thing (e.g., "Being pale always means sickness.", "If you have a headache, it must be a brain tumor.", "Yawning means you're always bored.")
Define <PD> to be a defective argument type making examples that fit oneself or others as well (e.g., criticizing marriage when you have remarried, Criticizing a fitness trainer for being a smoker, Calling out a financial advisor for not having a luxury car themselves.)
Define <B> to be a defective argument type offering overused excuses (e.g., "He did it because of anger.", "He lied because he was under a lot of stress.", "She didn't complete the project due to time constraints.")
Define <BA> to be a defective argument type presenting arguments too late in the timeline of the discourse (e.g., "We should have saved more money last year to have avoided this debt crisis.", "If you had started studying earlier, you wouldn't be in a panic now.")
Define <MD> to be a defective argument type using pretexts to cover evident faults (e.g., "I wasn't late; the clocks are just fast.", "I didn't cheat; I just had the notes open by accident.")
Define <MiD> to be a defective argument type offering false or overly general definitions (e.g., "Freedom is doing whatever you want whenever you want.", "Truth is whatever you believe in.")
Define <QS> to be a defective argument type using unproven assertions to support an argument (e.g., "Everyone who eats chocolate is healthier than those who don’t.", "Vaccines cause more harm than good.")
Define <InR> to be a defective argument type contradicting one's own arguments in the discourse (e.g., Arguing for environmental protection but supporting policies that exploit natural resources, Advocating for minimal government while simultaneously demanding government intervention in certain industries)
Define <HA> to be a defective argument type speaking against the audience’s beliefs (e.g., "If you’re smart, you wouldn’t need this explained to you.", "Only an idiot would disagree with me on this point.")
Define <IP> to be a defective argument type failing to substantiate all promised claims (e.g., Claiming that a certain diet leads to weight loss but not showing evidence for all cases, Asserting a product's effectiveness based on a single testimonial without scientific backing.)
Define <TS> to be a defective argument type deviating from the main issue under dispute (e.g., Diverting a discussion about climate change to talk about political ideologies, Switching a debate on healthcare to education systems.)
Define <FBA> to be a defective argument type disparaging a field due to the faults of its practitioners (e.g., Dismissing an entire scientific field because of a few fraudulent scientists, Generalizing negative traits found in a few individuals of a profession to all members.)
Define <EMU> to be a defective argument type proofing misconduct without identifying the perpetrator (e.g., Using circumstantial evidence to accuse someone without concrete proof, Blaming a group for the actions of one individual)
Define <BC> to be a defective argument type utilizing unfair or unbalanced comparisons (e.g., Comparing the life quality of a billionaire’s child to a middle-class child, Judging the performance of students from highly different educational backgrounds.)
Define <ND> to be a defective argument type creating controversy over naming conventions when usage should decide the matter (e.g., Arguing over whether a tomato is a fruit or vegetable instead of discussing its nutritional value, Debating the correct term for a couch versus a sofa when both are understood colloquially)
Define <IS> to be a defective argument type using inaccurate comparisons (e.g., Saying "this book is a rollercoaster" when it's moderately suspenseful at best, Comparing a minor issue to a world-ending catastrophe)
Define <UE> to be a defective argument type using examples that are false, inappropriate, or overextended (e.g., Using an isolated incident to generalize about a whole group, Claiming universal application from a single case study)
Define <MJ> to be a defective argument type referencing irrelevant or inappropriate previous judgments (e.g., Referencing a legal case that's irrelevant to the current case, Using outdated or contextually different rulings to support your stance)
Define <UnE> to be a defective argument type proofing points that adversaries concede (e.g., Proving the gravity of Earth's existence in a debate about specific falling objects, Arguing that fire is hot when everyone agrees already.)
Define <AE> to be a defective argument type amplifying points that should be substantiated, not because of their controversy (e.g., Focusing excessively on a trivial point to make it appear important, Elevating minor faults into major defects.)
Define <InS> to be a defective argument type concluding without covering all parts precisely (e.g., Ending a report without covering the results or conclusions sufficiently, Concluding an argument without addressing all key points discussed)
